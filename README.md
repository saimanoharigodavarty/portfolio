# Portfolio Website

A personal portfolio website for **Sai Manohari Godavarty**.

## Setup & Run

```bash
# Navigate to the project folder
cd portfolio

# Run the development server
python app.py
```

The server starts at **http://localhost:5000** and opens automatically in your browser.

## Project Structure

```
portfolio/
├── app.py          # Python development server
├── index.html      # Main HTML file
├── styles.css      # Stylesheet (light professional theme)
├── script.js       # Interactive JavaScript
└── README.md       # This file
```

## Deployment

This is a static site. Deploy to any hosting platform:
- **GitHub Pages** – Push to a `gh-pages` branch
- **Netlify** – Drag & drop the folder
- **Vercel** – Import the repository
- **Render** – Connect the GitHub repo

## Tech Stack

- HTML5 / CSS3 / JavaScript (Vanilla)
- Python (development server)
- Google Fonts (Inter, JetBrains Mono)
- Font Awesome 6 (icons)
